# Requirements

This project will be a competitive programming progress tracker, recommender, and notifier.

It will:

- Allow a user to keep track of and manage problems solved
- Share a status update via notification
- Recommend problems to solve?
- Use microservices
- Use Java and Spring Boot.
- A Management interface
- Easy way to import stuff

**Problem Management**:

- Add problem
- Update problem
- Delete problem

**Recommender**:

- Show what to work on
- Let user determine weight of problem types

**Notifier**:

- Share breakdown of problems solved today, this week, this month, this year?
- Share what is recommended to work on next

**Importer**:

- CSV import?
- Image import?
- PDF import?
- Spreadsheet import?
- Some sort of platform integration?

**Management Interface**:

- TUI (for now) to make it easy to manage stuff, quickly.

**Analytics**:

- Problems solved on X day
- Problems solved this week compared to last week
- Problems solved this month
- Trends
- Most active users

## Priorities

1. Problem Management
2. Testing
3. Management Interface
4. Notification System
5. Recommendation System
6. Logging
7. Importer
8. Analytics

MVP will contain 1–3.

Deadline: 1 week.
